Aria2 配置与使用

Posted on 2019-01-22

aria2 is a lightweight multi-protocol & multi-source command-line download utility. It supports HTTP/HTTPS, FTP, SFTP, BitTorrent and Metalink. aria2 can be manipulated via built-in JSON-RPC and XML-RPC interfaces.

配置

创建配置文件 ~/.aria2/aria2.conf 与进度保存文件 ~/.aria2/aria2.session。

配置文件填写可以查看Aria2 & YAAW 使用说明，可以在配置文件中指定下载位置、指定是否启动 RPC 等。

配置示例下载

使用

直接下载

直接下载正如字面，可以使用命令 aria2c "http://host/file.zip" 直接下载文件，在下载完成后进程退出。

RPC Server 模式

在该模式下，等待从 RPC 接口添加任务，下载完成后继续等待。

使用命令启动

使用命令 aria2c --enable-rpc --rpc-listen-all=true --rpc-allow-origin-all -c -D 可以启动 RPC Server，但是这样做不好修改配置，-D 后台运行也看不到出错的信息，推荐使用配置文件启动的方式。

使用配置文件启动

我们之前已经创建好了配置文件，并填好了相关配置。

使用 aria2c --conf-path=<PATH> 便可以按照配置文件中的配置启动 RPC server，可以添加 -D 后台运行。

启动后可以访问 Aria2 Web 控制台 YAAW，按照 Aria2 & YAAW 使用说明填写正确的 JSON-RPC Path 后，就可以在 Web 控制台添加任务了。

Aria2 的 Web 控制台有不少，可以选择自己喜欢的使用，比如：

  • YAAW
  • WebUI-Aria2
