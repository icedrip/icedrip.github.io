西电 LaTeX 模板使用说明

Posted on 2018-12-22 | Edited on 2018-12-26

LaTeX，是基于 TeX 的排版系统，对与任何有复杂排版的使用场景，它都可以生成书籍质量的文档，这一点在数学及科技类文档中的表现非常突出。简而言之，LaTeX 是极佳的论文书写及排版工具。

概览

LaTeX 可以类比为 HTML 网页。类似网页一个 .html 文件控制文档结构，一个 .css 文件控制文档样式，LaTeX 中的 .tex 文件用来控制文档结构，.cls .sty 文件控制样式。

LaTeX 模板常见的文件类型有很多：

文件类型 功能
.dtx Documented LaTex sources，宏包重要部分
.ins installation，控制 TeX 从 .dtx 文件里释放宏包
.cfg config，配置文件，可由上面两个文件生成
.sty style files，使用 \usepackage{...} 命令加载
.cls classes files，类文件，使用 \documentclass{...} 命令加载
.aux auxiliary，辅助文件，不影响正常使用
.bst BibTeX style file，用来控制参考文献样式

安装 LaTeX

选择一个 TeX 发行版安装 LaTeX 环境，以 MacOS 为例，可以在这里下载安装 MacTeX，安装包比较大，足足有大约 3.2G （17 April 2018），完整安装会让你省心不少。

完成安装后，Applications 目录中会出现 5 个软件：BibDesk、LaTeXiT、TeX Live Utility、TeXShop、Excalibur。

其中 TeXShop 是 MacTeX 发行版自带的 LaTeX 编辑器。

可见文件并非 TexShop 默认打开文件时用的 utf-8 编码，ISO-8859 是一个字符集，其并不能表示中文，但是配合电脑自己的其他编码可以正确显示中文。

修改 TexShop -> 偏好设置 -> 源代码 -> 编码 为 GBK，Command-Q 退出 TexShop，重新打开文件后，可以正常显示中文。但是想把默认编码改为 utf-8 。

103yiran/XDUthesis_xelatex

缺少字体

首次编译，MacOS 下缺少 SimSun（中易宋体）、SimHei（中易黑体）两款字体，安装对应字体后问题解决。

引用显示不正确

再次编译，引用论文时角标没有正常显示（空白或无意义字符），更新 TeXShop 至 4.21 版本后问题解决。

总结

TeX 众多的发行版方便了 LaTeX 环境的搭建，主要是宏包的选择问题。

西电研究生院提供的宏包较为老旧，存在很多小问题，即便能够通过编译了，正文还需要做一些修改，比如中文 ～ 符号就无法如期显示。

103yiran 修改过的版本则好用很多，除了字体缺失，其他没有什么问题，十分推荐使用。
