启动 Accumulo

有几种启动方法：

  1. 使用 accumulo 命令在前台启动 processes，这不会重定向 stderr/stdout。
  2. 使用 accumulo-service 以服务启动 processes，其使用 accumulo 命令在后台启动 processes，重定向 stderr/stdout 并管理 pid files。
  3. 使用 accumulo-cluster 在一个或多个节点启动 cluster。适合本地开发/测试或在没有使用 cluster 管理工具的生产环境。

启动 Accumulo cluster

需要创建额外的配置文件。

这在会在 conf/ 目录下创建五个文件（master, gc, monitor, tservers, tracers），其包含了 Accumulo 服务运行所在的 node names。默认所有文件都设置为 localhost，如果你在单节点运行 Accumulo cluster，这些文件不用修改。
